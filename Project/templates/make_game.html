Make Game

This is the DEMO or prototype build of the game, the purpose of which is to provide an idea of how the core gameplay loop would feel. It functions by simply having a pool of quizzes to choose from. To prevent the pool from being filled with many test quizzes from all the potential testers, we chose to simply provide a predefined list of quizzes that can be used to create a game. If you wish to see the quiz creation windows as well, please have a look at our project on GitHub.
